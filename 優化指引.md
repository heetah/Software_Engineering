### 關於OpenAI 和 Gemini API
1. 要自行測試時可以透過修改 **.env.example** 將其中的 *OPENAI_API_KEY* 或 *GEMINI_API_KEY* 替換成自己的 API
2. 只要填寫一個 API 即可，OpenAI API 要先儲值 $5，而 Gemini API 可以免費使用，但注意 免費額度很低，但可以透過**不斷創造新的API KEY（要在不同project中）** 來無限使用

### 提交限制
1. 盡量以調整自己負責的部分為主
2. 一樣只能 commit 到自己的 branch 
3. 如果需要整合可以在群組或私訊講一聲
