使用說明

快速開始 (Quick Start)

初次使用請參考以下步驟完成設定。

前往「設定」頁面，輸入您的 Gemini 或 OpenAI API Key。
注意：您的 Key 僅儲存於本地端，不會上傳至任何外部伺服器。

預設使用 "Auto" 模式（優先嘗試 Gemini）。您可以在設定中強制指定使用特定模型。

功能介紹 & 小技巧

了解如何與 AI Copilot 協作。

像與真人工程師對話一樣，詢問程式碼問題、要求重構或 Debug。

AI 能讀取您當前的工作區檔案。當您提到「這個檔案」或具體檔名時，AI 會自動分析相關內容。

在「設定」頁面中開啟深色模式，不僅保護眼睛，還能體驗更沉浸的介面風格。

按下 F12 可開啟開發者工具 (DevTools)，方便進階使用者除錯與檢視程式狀態。

按下 Ctrl + R (macOS 為 Cmd + R) 可立即開啟一個全新的對話窗口。

疑難排解與常見疑問。

請確認您有權限寫入目標資料夾，且路徑中沒有特殊字元。若問題持續，請檢查 Console 下的錯誤訊息。
